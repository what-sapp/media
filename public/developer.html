📖 Code Examples

cURL:
curl -X POST https://your-domain.com/api/v1/download \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://youtube.com/watch?v=..."}'
JavaScript (Fetch):
fetch('https://your-domain.com/api/v1/download', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ url: 'https://youtube.com/watch?v=...' })
})
.then(res => res.json())
.then(data => console.log(data));
Python:
import requests

headers = {'Authorization': 'Bearer YOUR_API_KEY'}
data = {'url': 'https://youtube.com/watch?v=...'}
response = requests.post('https://your-domain.com/api/v1/download', headers=headers, json=data)
print(response.json())
